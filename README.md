# solixo
Solixo will be an app for tracking job applications. The primary goal of this project is to explore Java Spring Boot, Jakarta EE, Docker, Angular, and PostgreSQL, which are technologies that are widely used among companies.

## back-end

### api-gateway
Built using Spring framework. The selected dependencies are Reactive Gateway, Resilience4J, and Contract Stub Runner like suggested in the spring gateway guide.

### databse
At the moment, it is just a docker file to run a postgreSQL database in a docker container.

## front-end
Default front-end from Angular without SSR.
